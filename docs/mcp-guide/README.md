## 快速理解

```
┌─────────────┐      ┌─────────────┐      ┌─────────────┐
│   AI 客户端  │ ←──→ │   MCP 服务器  │ ←──→ │  外部资源    │
│ (Claude等)  │      │             │      │ (文件/API等) │
└─────────────┘      └─────────────┘      └─────────────┘
```

**一句话总结**：MCP 就像 AI 界的 USB 接口标准——统一的插口，无限的可能。
